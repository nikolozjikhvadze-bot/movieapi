# movieapi

# 🎬 Movie API - ფილმების მონაცემთა ბაზა და მართვის სისტემა

ეს არის **Movie API** — სერვერული აპლიკაცია, რომელიც უზრუნველყოფს ფილმების, ჟანრების, მსახიობებისა და მომხმარებლების შეფასებების მართვას. პროექტი აგებულია REST API-ს პრინციპებით და საშუალებას გაძლევთ მარტივად განახორციელოთ ფილმების ძებნა, ფილტრაცია და ადმინისტრირება.

---

## ✨ ძირითადი ფუნქციონალი (Features)

* **🎬 ფილმების კატალოგი (Movies CRUD):** ფილმების სრული ბაზა — დამატება, წაშლა, რედაქტირება და წაკითხვა (სათაური, აღწერა, გამოშვების წელი, რეჟისორი, პოსტერი).
* **🔍 გაფართოებული ძებნა და ფილტრაცია:** ფილმების მოძებნა სათაურის, ჟანრის (Genre), რეიტინგისა და გამოშვების წლის მიხედვით.
* **⭐ შეფასებების სისტემა (Reviews & Ratings):** ავტორიზებულ მომხმარებლებს შეუძლიათ ფილმებზე რეიტინგის (1-დან 5-მდე) დაწერა და კომენტარის დატოვება.
* **🔐 როლების მართვა (RBAC):** ჩვეულებრივ მომხმარებლებს მხოლოდ ფილმების ნახვა და შეფასება შეუძლიათ, ხოლო ახალი ფილმების დამატება/წაშლა მხოლოდ **ადმინისტრატორის (Admin)** პრივილეგიაა.

---

## 🛠️ ტექნოლოგიური სტეკი

* **Runtime Environment:** Node.js
* **Framework:** Express.js
* **Database:** MongoDB / Mongoose ODM (ან შესაბამისი ბაზა, რომელსაც იყენებ)
* **Authentication:** JWT (JSON Web Tokens) & bcrypt პაროლების ჰეშირებისთვის.

---

## 🛣️ API ენდპოინტები (Routes)

### 🍿 ფილმები (`/api/movies`)
* `GET /api/movies` — ყველა ფილმის სიის წამოღება (ფილტრაციით).
* `GET /api/movies/:id` — კონკრეტული ფილმის დეტალები.
* `POST /api/movies` — ახალი ფილმის დამატება *(მხოლოდ ადმინი)*.
* `PUT /api/movies/:id` — ფილმის ინფორმაციის განახლება *(მხოლოდ ადმინი)*.
* `DELETE /api/movies/:id` — ფილმის წაშლა ბაზიდან *(მხოლოდ ადმინი)*.

### 👤 მომხმარებლები და შეფასებები (`/api/users` / `/api/reviews`)
* `POST /api/users/register` — რეგისტრაცია.
* `POST /api/users/login` — სისტემაში შესვლა (ტოკენის მიღება).
* `POST /api/reviews/:movieId` — ფილმზე შეფასების დატოვება.
